# cshaprmassiv

Учебный консольный проект на C# для практики массивов, циклов (`for`), условий (`if`) и базового ввода/вывода.

> Репозиторий содержит решение Visual Studio (`cshaprmassiv.sln`) и проект с заданиями. 

## Что внутри

- Ввод массива с клавиатуры
- Обработка массива:
  - поиск минимума/максимума
  - сумма/среднее (по желанию)
  - вывод массива в обратном порядке
- Небольшие задания (tasknumber1, tasknumber2 и т.п.)

## Требования

- .NET SDK (желательно .NET 6/7/8)
- Visual Studio 2022 / Rider / VS Code (с C# расширением)

## Как запустить

### В Visual Studio
1. Открой cshaprmassiv.sln
2. Выбери проект как Startup Project
3. Нажми Start (F5) или Ctrl+F5

### Через терминал (если удобно)
1. Открой папку с проектом в терминале
2. Выполни:
   ```bash
   dotnet run
